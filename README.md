# 🚀 Alexis Marketing & Desenvolvimento - Site Oficial

Este é o repositório da plataforma oficial da **Alexis Marketing & Dev**. O site serve como nosso portfólio principal e demonstra nossas capacidades em engenharia de software, design exclusivo e arquitetura digital de alta performance.

## 🛠️ Tecnologias de Ponta
* **Frontend:** Interface desenvolvida para ser rápida, responsiva e focada em conversão.
* **Segurança Avançada:** Implementação de políticas de RLS (Row Level Security) para proteção de dados.
* **Sistemas Integrados:** Módulos de contrato com login seguro e validação por Captcha.
* **Otimização de Conversão:** Fluxos separados para download e compra, maximizando a experiência do usuário.

## 📈 Foco em SEO e Performance
* **SEO Nativo:** Estrutura de metatags e semântica HTML otimizada para dominar os resultados de busca.
* **Arquitetura Base44:** Implementação de padrões de chave de plano para escalabilidade de software.
* **Componentização:** Landing pages construídas com componentes modulares e reutilizáveis.

## 📂 Estrutura do Projeto
* `/src`: Lógica central e páginas da aplicação.
* `/components/landing`: Componentes específicos para páginas de destino de alta conversão.
* `/directives`: Diretivas personalizadas para comportamentos de UI e segurança.
* `/base44`: Módulos de lógica de negócios e planos de software.

## 🚀 Versão Atual
O projeto encontra-se na versão **v2.9.0**, incluindo as últimas atualizações de textos de marketing e melhorias de segurança nas políticas de RLS.
